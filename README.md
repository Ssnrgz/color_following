HSV değerleri kullanılarak renkli nesnelerin tespiti ve takibi yapılmaktadır. Nesne tespit edilemediğinde robot kendi etrafında dönmektedir. Tespitin sağlanabilmesi için HSV değerlerini değiştirmeniz gerekmektedir. Tespit gerçekleştiği an robot nesneye doğru ilerleyecektir. 
İyi eğlenceler!!

Detection and tracking of colored objects using HSV values. When no object is detected, the robot rotates on its own axis. You must change the HSV values to enable detection. The moment detection occurs, the robot will move towards the object. 
Have fun!!
